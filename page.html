Manual Mode

Insert the solution into the text box below and click enter.

The solution should be a comma separated list of lists of 0s and 1s, where 0 is empty and 1 is filled.

Make sure all the lists are the same length.

Or type in #x# to get an empty grid of the specified size.

Press the export button to display the current board in the text box.
